# DashboardApp

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.0.5.

## Development server

### PROJECT SETUP 
```
- clone the repo
    git clone git@github.com:kelvinndmo/DashboardApp.git

- install dependencies
    npm install

- Run the development server
    ng serve
```
